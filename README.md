# Around The World

A small React app that lists countries, allows searching/filtering and shows country details using the REST Countries API. Built with Vite, React and Tailwind CSS.

## Features
- Browse countries with flag, population, region and capital
- Search by country name and filter by region
- Country details page (languages, currencies, borders)
- Light / Dark theme toggle persisted to localStorage
- Caches the last successful country list in localStorage for offline fallback
